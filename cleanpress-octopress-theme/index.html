CleanPress Octopress Theme

I’m a huge fan of clean, white, elegant and responsive websites so when I originally set up this blog I created my own custom design based on the Slash theme but taking a lot of influence from Guillermo Esteves beautiful blog while still retaining an element of originality.

The theme is now at a point where I’m happy to share it publicly, so I’ve given it a name of ‘CleanPress’ and made it available via my GitHub repository.

There’s still work to do in cleaning up the remnants of the Slash Theme and I’m no expert on the Octopress platform as yet so there’s no doubt much mess but I’m hoping someone can take it on and improve it further.

In the meantime, I’ll happily take any comments or questions on the current version and if you do use it on your own site or modify it further I’d love to know.
