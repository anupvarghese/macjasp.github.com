This site will focus primarily on photography occasionally interspersed with pieces of code and other technical ramblings.

About this Site (Colophon)

This site is a “baked blog” running on the quite lovely Octopress with content generated on a Raspberry Pi and hosted on the equally adorable Github Pages.

Images are hosted via Dropbox and Flickr.

The theme is my own modification of the Slash theme with the Proxima Nova font provided for free by Typekit.

All content is written in Markdown using iA Writer.

Changes I’ve made to date are;

  • Removed the /blog/ subdirectory and date from the permalink path, going with a cleaner domain/post-title format.

  • Installed Felipe Cypriano’s Footnotes plugin.

  • Installed Scott W. Bradley’s Tweet Tag.

  • Installed my own forked version of tsmango’s Flickr Set Plugin

About Me

I’m an IT Consultant, self confessed tech geek and when I can be bothered to, an occasional developer in iOS and web technologies.

I have a passion for minimalistic, elegant design. I listen to melancholic / kitchen sink music endlessly, carry a camera with me everywhere and when the mood suits, I run.
