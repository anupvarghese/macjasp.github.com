About this Site (Content)

This site will focus primarily on photography occassionally interspersed with pieces of code and other technical ramblings.

About this Site (Technical)

This site is a “baked blog” running on the quite lovely Octopress with content generated and hosted on the equally adorable Raspberry Pi.

Images are hosted via Dropbox and Flickr.

The theme is my own modification of the Slash theme with the Proxima Nova font provided for free by Typekit.

All content is written in Markdown using iA Writer.
