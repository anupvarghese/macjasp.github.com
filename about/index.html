About this Site (Content)

This is my own personal blog primarily designed to demonstrate the Cleanpress theme for Octopress.

About this Site (Colophon)

This site is a “baked blog” built on the quite lovely Octopress with content generated on a Raspberry Pi and hosted on the equally adorable GitHub Pages.

Images are hosted via Dropbox and Flickr.

The theme is called Cleanpress with the Proxima Nova font provided for free by Typekit.

All content is written in Markdown using iA Writer.

Changes on this site to date are;

  • Removed the /blog/ subdirectory and date from the permalink path, going with a cleaner domain/post-title format.

  • Installed Felipe Cypriano’s Footnotes plugin.

  • Installed Scott W. Bradley’s Tweet Tag.

  • Installed my own forked version of tsmango’s Flickr Set Plugin
